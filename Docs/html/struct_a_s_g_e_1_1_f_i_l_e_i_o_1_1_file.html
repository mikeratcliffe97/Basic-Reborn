ASGE::FILEIO::File struct

a file stored locally

Contents

  • Reference
    • Public functions

A file can be used to open a resource. It supports reading and writing and provides an abstraction between the low-level IO libraries. Reading from a file will generate an IOBuffer object, which can be used to load resources such as fonts.

buffer The data to be written.

IOBuffer ASGE::FILEIO::File::read()

Returns The data buffer.

Reads the contents of the file. Stores the entire contents of the open file into a buffer. The buffer stores both the data and its length.
